## background-color
- Class prefix: `b`
- Values: `<HEX>` / `<#EAEAEA>`

The background-color CSS property sets the background color of an element.

### Syntax
```html
<!-- A square gray square with a text inside "sets background" -->
<div class="b-EAEAEA h-100px w-200px"> sets background</div>
 ```
